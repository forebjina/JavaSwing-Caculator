# JavaSwing으로 구현한 계산기 어플





### 주요기능
1. 기본적인 사칙연산기능 구현('+', '-', 'x', '/')
2. '.'입력시 double형 사칙연산 기능 구현
3. 'C'버튼 클릭시 초기화
4. '←'버튼 클릭시 한 자리씩 지우기(한 자리만 남았을 경우 '0'으로 초기화)
